## Focused versus Diffuse Thinking

* Focused mode와 Diffuse mode

![image](/learning-how-to-learn/images/1.png)

---

### Focused mode (집중 모드)

* 핀볼의 좁은 공간의 고무범프처럼 우리의 생각을 집중하는 상태
* 간단한 일부터 복잡한 일까지 문제를 이해하기 위한 모드. 전두엽에 집중.
* 예) 곱셈, 단어 라임

---

### Diffuse mode (분산 모드)

* 핀볼의 넓게 배치된 고무범프처럼 좀더 넓은 사고를 하는 상태
* 새로운 문제 해결, 더 넒은 관점 필요, 뇌 휴지 상태
* 산책, 샤워, 잠들기 전
* 어려운 문제, 난처한 상황
* 집중했다가 그냥 두고 여유를 가지면 의식못하는 사이 Diffused mode에서 뇌 활동

### 미루기

* 하고 싶지 않은 일을 보고 있을 때 고통 관련 뇌 영역 활성화, 일단 시작하면 불편함이 사라짐
* 미루기의 결과로 (시험 전)벼락치기 공부
* 뇌속 새로운 신경구조는 단기간에 형성 안되므로, 형성위해 꾸준함이 중요
* 연습과 반복을 통해 새로운 것 배울 때 신경 구조를 확장하고 강화할 수 있음
* 이를 위해 뽀모도로 기법을 사용해보자.

### 단기 기억과 장기 기억

* 단기 기억을 위해 4개의 자리가 있고, 반복하지 않으면 지워짐
* 반복을 통해 장기 기억 영역에 저장하면 나중에 필요시 기억
* 기억을 새롭게 하기 위해 주기적 반복이 필요. 하루에 여러번 반복은 안좋음.(하루에 근육만들기)

### 잠의 중요성

* 낮에 쌓이 독성 물질을 씻어내는 역할
* 자는 동안 어려운 부분을 깊게 만들고 강화하는 연습을 함 (Focused mode --> Diffused mode)
* 시험이나 어려운 일 전에는 충분히 자야 함

### Chunk

* 정보의 단위, 이용과 의미 부여를 통해 묶임
* 중요한 생각 또는 행동을 종합하는 신경체계
* 더크고 복잡해질 수 있으며 쉽게 접근하게 하는 단일체
* 제한된 작업 기억 공간을 차지 하지 않고 chunk에 쉽게 접근할 수 있음

![image](/learning-how-to-learn/images/8.png)

### How to Form a Chunk

* 기초적인 이해, 집중과 지속적 관심, 신경형상을 짙게하고 큰그림을 만드는 연습으로 잘 만들어짐
* 책 보지 않고 주요사항을 기억하려는 회상(recall)은 chunking 과정을 돕는 가장 좋은 방법
* 한 영역의 숙련된 chunk는 다른 영역의 chunk를 쉽게 배울 수 있게 하며 이를 "전이"라 함
* 한 기간에 다른 개념, 접근법, 기법으로 연습함으로 interleave 하라
* 하지만, chunk는 유연성 형성과는 관계가 없으며, 유연성은 전문가가 되는데 중요

### Illusions of Competence

* 학습에 대한 자신감 환상 (Illusions of competence in learning)
* 학습시 스스로를 기만하는 때를 인지해야 함
* 미니 시험과 회상. 회상은 미니시험의 한 형태
* 실수는 자신감 환상을 알아챌 수 있게 함
* 쉬운 것만 연습하는 것(Overlearning)을 회피해야. 숙련했다는 환상을 만듦.
* 보다 어려운 것을 찾아 신중히 연습 ([Deliberate Practice](https://jigjang.wordpress.com/2016/07/05/deliberate-practice/))

---

### Einstellung

* Einstellung - \[|ain∫tεlʊŋ], 사고방식, mindset
* 최초의 생각, 이미 습득한 방법, 이미 잘 발전&강화한 신경형상이 더 나은 방법 새로운 해결책을 수용하도록 유연해지는 것을 방해
* 운좋은 발견의 법칙(serendipity)은 유용. 새로운 시도. 항상 다른 것을 선택하면 좋은 결과

![einstellung](/learning-how-to-learn/images/einstellung.png)

### Interleaving

* Interleave - v. 끼우다.
* 새로운 주제를 숙달하기 위해 서로 다른 기법 또는 전략을 요구하는 문제 또는 상황을 왔다 갔다 하는 연습을 하는 것
* 예들어 시험을 대비시 서로 다른 장과 자료 속에 있는 문제들을 가볍게 훑어보면서 검토하는 것이 심도있게 학습하도록 도와줍니다.
* 연습과 반복은 견고한 신경 형상에 중요, 유연성과 창의성을 형성에는 interleaving이 도움
* Interleave 하면 다른 분야의 chunks 간 흥미로운 새로운 연결로 창의력 강화

---

### Procrastination

* 좋은 학습은 조금씩 매일 견고한 발판을 쌓는 것과 관련 (역도선수 매일 근육 만들기)
* 미루기 방지가 이래서 중요. 학습을 지속하고 벼락치기를 피해야.
* 일정표, 목표 도달시 완료 항목을 관찰
* 매일 일정한 반복된 일상과 업무에 전념
* 업무를 완수할 떄까지 일부러 보상을 미룰 것
* 매일 먼저 개구리를 먹기

### Memory

* 장기 기억 (Long Term Memory)
	* 저장 창고. 연습과 반복이 필요. 쉽게 회상
	* 일찍 시작, 여러 날에 걸쳐 연습. 미루기 방지가 중요

* 작업 기억 (Working Memory)
	* 낡은 칠판. 4개의 목록만 저장
	* 기술과 개념을 숙달하여 청크화하면 작업 기억 공간을 차지 않음
	* 회상시에 정신적 생각 공간을 비우게하고 다른 생각을 쉽게 이해할 수 있게 함

---

### Memory 기법

* 시각적 기억 체계 이용 위해 인상적인 시각적 이미지를 만들고, 보는 것 넘어 느끼고 듣고, 냄새까지
* 기억의 중요한 것은 자료를 단순화한 의미있는 무리를 만드는 것
* 훈련을 통해 기억할 문장, 숫자를 다른 것에 연결
* 기억을 무리화. 의미있는 무리 또는 약어 만들기
* 배울 것을 단순화하고 chunk할 수 있고 쉽게 기억 저장

### Summary

* 집중하여 문제를 이해하기 위해서 - Focus mode, 뽀모도로 기법
* 새로운 문제, 어려운 문제 해결을 위해서 - Diffuse mode
* 잘 이해하고 잘 수행하기 위해 - chunk, recall, Einstellung, Interleaving, Deliberate Practice
* 미루기 방지 - 뽀모도로 기법, Habits have four parts(cue, routine, reward, belief)
* 잘 기억하기 위해 - Memory Palace, chunk, 에빙하우스의 망각곡선
